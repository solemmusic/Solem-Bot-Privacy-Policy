# 🔹 Solem Music Bot – Gizlilik & Veri Politikası / Privacy & Data Policy

- Kullanıcı ID (User ID)  
- Sunucu adı (Server name)

- Sunucu üye sayısı (Number of members)

Hizmet sunumu, güvenlik, performans takibi ve destek
